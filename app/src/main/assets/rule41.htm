Правило 41.

Проверка соответствия

Участники применяют положения Кодекса по осуществлению при выполнении своих обязательств и обязанностей, содержащихся в настоящей Конвенции.

  1. Каждый Участник подвергается периодическим проверкам со стороны Организации в соответствии со стандартом проверки для проверки соответствия настоящей Конвенции и ее осуществления.

  2. Генеральный секретарь Организации несет ответственность за осуществление Системы проверки на основе руководства, разработанного Организацией*.

  3. Каждый Участник несет ответственность за облегчение проведения проверки и осуществление программы принятия мер в свете сделанных выводов на основе руководства, разработанного Организацией*.

  4. Проверка всех Участников:

    1. основана на общей системе, разработанной Генеральным секретарем Организации, с учетом руководства, разработанного Организацией*;

    2. проводится с регулярными интервалами с учетом руководства, разработанного Организацией*».

* См. Рамочные принципы и процедуры Системы проверки государств-членов ИМО (резолюция A.1067(28)).
